# Default Assign Result Function

Used internally in the
[Optimizer](https://bbotk.mlr-org.com/reference/Optimizer.md). It is the
default way to determine the result by simply obtaining the best
performing result from the archive.

## Usage

``` r
assign_result_default(inst)
```

## Arguments

- inst:

  [OptimInstance](https://bbotk.mlr-org.com/reference/OptimInstance.md)
