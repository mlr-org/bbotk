# Calculate the transformed x-values

Transforms a given [`list()`](https://rdrr.io/r/base/list.html) to a
list with transformed x values.

## Usage

``` r
trafo_xs(xs, search_space)
```

## Arguments

- xs:

  ([`list()`](https://rdrr.io/r/base/list.html))  
  List of x-values.

- search_space:

  [paradox::ParamSet](https://paradox.mlr-org.com/reference/ParamSet.html)  
  Search space.

## Value

[`list()`](https://rdrr.io/r/base/list.html) with transformed x values.
